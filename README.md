# Chinese Chess Game

About
-----

Implemented by *C++*

Pre-requirement
---------------

* Linux OS: For epoll

* g++ >= 4.8.5

* make >= 3.82

Build and Run
-------------

To build the project:

    % make build

To start the Server:

    % make run-server

To start the Client:

    % make run-client

Default:

* TCP Port: "8889"

* Server IP: "127.0.0.1"
